**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Orientation, environment setup, and FCJ introduction](1.1-week1/)

**Week 2:** [Team onboarding, reading unit rules, and initial project familiarization](1.2-week2/)

**Week 3:** [Learn AWS service categories (Compute, Storage, Networking, Database) and foundational concepts](1.3-week3/)

**Week 4:** [Create AWS Free Tier account, configure AWS CLI, and practice basic EC2 usage](1.4-week4/)

**Week 5:** [AI integration planning, MSSQL tasks, and project design updates](1.5-week5/)

**Week 6:** [Implement Cognito auth flows, Spring Data JPARepository work, and fix backend errors](1.6-week6/)

**Week 7:** [Migrate from RDS SQL Server to DynamoDB and implement admin-side CRUD](1.7-week7/)

**Week 8:** [OJT test preparation and Coursera specialization completion](1.8-week8/)

**Week 9:** [Continue DynamoDB migration, admin CRUD improvements, and Linux/EC2 commands practice](1.9-week9/)

**Week 10:** [Implement user-side CRUD, Phuc Long team meeting, and prototype Bedrock Chatbot integration](1.10-week10/)

**Week 11:** [Add credit-card payment flows and attend AWS Cloud Mastery Series (Bitexco)](1.11-week11/)

**Week 12:** [Project bug review, deploy backend/database/frontend to AWS, and demo preparation](1.12-week12/)
